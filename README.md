# Installation

To add cogs from this repo to your instance, do these steps:

- `[p]repo add cogs https://github.com/Honkertonken/Cogs`
- `[p]cog install cogs <cog name>`
- `[p]load <cog name>`

`Note: [p] here refers to your prefix.`
